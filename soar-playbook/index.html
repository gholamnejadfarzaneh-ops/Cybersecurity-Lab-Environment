SOAR Playbook: Automated URL Triage

Security Automation / Python / VirusTotal API

In modern Security Operations Centers (SOC), speed is a defensive necessity. This project demonstrates a SOAR (Security Orchestration, Automation, and Response) playbook designed to automate the initial triage of suspicious URLs.

The Logic

Using Python, the script interacts with the VirusTotal API to scan submitted URLs. Instead of a manual search, the playbook automatically categorizes the threat level based on engine detections.

Code Snippet

import requests

def triage_url(url_id, api_key):
    headers = {"x-apikey": api_key}
    response = requests.get(f"https://www.virustotal.com/api/v3/urls/{url_id}", headers=headers)
    return response.json()
